# Лабораторная работа 2

Тема: потоки, процессы, асинхронность.

## Структура

- `task1_sum/` — сравнение подходов на вычислении суммы от `1` до `10_000_000_000_000`.
- `task2_parsing/` — параллельный парсинг страниц и запись результатов в БД ЛР1.

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Настройка окружения

Создайте `.env` в папке `Lr2` на основе `.env.example`:

```env
DB_URL=postgresql://postgres:123@localhost/time_manager_db
```

## Быстрый сценарий проверки

1. Запустите все варианты из `task1_sum` и сравните время.
2. Запустите все варианты из `task2_parsing` и убедитесь, что данные попали в таблицу `tag`.
3. Внесите фактические времена в документацию MkDocs.
